# United Nurse Registery
the last registery to rule them all.

##### install
```sh
npm install
```

##### run development build
```sh
npm run development
```

##### run production build
```sh
npm run production
node server
```

#### auto restart gulp
```sh
npm i -g gulper
gulper
```
