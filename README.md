# swiftcard
loyalties and more for free.

##### install
```sh
npm install
```

##### run development build
```sh
gulp
```

##### run production build
```sh
gulp production
node server
```

#### auto restart gulp
use gulper, it is nodemon but for gulp, it replaces gulp and gives you a restart ability by watching your gulp task changes.
```sh
npm i -g gulper
gulper production
gulper
```
